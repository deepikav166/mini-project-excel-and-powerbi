# Customer Loan Approval and Risk Analytics in Banking Sector

## 📌 Project Overview

This project focuses on analyzing customer loan approval and identifying risk levels using Excel and Power BI.
The main objective is to help banks understand customer financial behavior and make better loan approval decisions.

---

## 📊 Tools Used

* Microsoft Excel
* Power BI
* Power Query
* DAX

---

## 📁 Dataset Information

The dataset contains customer financial and personal details such as:

* Age
* Annual Income
* Credit Score
* Employment Status
* Loan Amount
* Debt-to-Income Ratio
* Savings Account Balance
* Interest Rate
* Loan Approval Status
* Risk Score

Total Records: 2648
Total Columns: 20

---

## 🔧 Data Preprocessing

The following preprocessing steps were performed:

* Removed duplicate records
* Checked missing values
* Formatted data types
* Created calculated columns
* Feature engineering using Power Query and DAX

---

## 📈 Dashboard Insights

Some important insights identified from the analysis:

* High-risk customers were identified based on risk score.
* Customers with low credit scores had a higher loan rejection rate.
* Higher debt-to-income ratio increased loan risk.
* Employment status and income level influenced loan approval decisions.

---

## 📊 Power BI Visualizations

The dashboard includes:

* Loan Approval Analysis
* Risk Score Analysis
* Income vs Loan Amount
* Credit Score Distribution
* Employment Status Analysis
* Interactive Filters and Slicers

---

## 🎯 Project Outcome

This project helps:

* Identify high-risk loan applicants
* Improve loan approval decisions
* Analyze customer financial patterns
* Support data-driven banking decisions
